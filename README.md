⚡ AI Study Assistant (RAG)
An interactive Retrieval-Augmented Generation (RAG) application for Electrical Engineering exam preparation.
It answers questions using your own notes and presents them in a structured, exam-friendly format.

🚀 Features
	•	📚 Context-based answers from personal notes
	•	🧠 RAG pipeline (retrieval + generation)
	•	🧾 Exam-oriented format (Definition, Formula, Explanation)
	•	📍 Source page references
	•	💬 Interactive chat UI (Streamlit)
	•	🎯 Multiple answer modes (normal, 5-mark, revision)

🛠️ Tech Stack
	•	Python, Streamlit
	•	LangChain (RAG pipeline)
	•	FAISS (vector search)
	•	Sentence Transformers (embeddings)
	•	Groq LLM

  🧠 How It Works
	1.	Notes are converted to text (rag.txt)
	2.	Text is split into chunks
	3.	Embeddings are created
	4.	Relevant chunks are retrieved
	5.	LLM generates answers using context
  
📌 Use Cases
	•	Exam preparation
	•	Quick revision
	•	Concept understanding
